# Skills Repository

本仓库包含从《美国资产管理行业的历史性变革》研究中提炼的7个方法论Skills。

## Skills列表

1. **行业变革分析框架** - 分析任何行业历史性变革的系统化方法论
2. **战略定位三维度** - 帮助企业在变革环境中找到差异化定位
3. **需求转化框架** - 将终端用户的隐性需求转化为可操作的产品功能
4. **组织能力重组五维度** - 帮助企业系统性评估并重构组织能力
5. **竞争模式四分法** - 帮助企业识别所在行业的竞争模式
6. **可行性三时间轴** - 评估战略举措的可行性，并按照短中长期时间轴分解
7. **科技投入优先级** - 帮助企业在资源有限的情况下，科学决定科技投入的优先顺序

## 使用方法

每个Skill都是一个独立的文件夹，包含`SKILL.md`文件。你可以：

1. 直接阅读`SKILL.md`学习方法论
2. 使用Skill中的模板和工具进行分析
3. 将Skill导入到支持的系统（如WorkBuddy）

## 背景

这些Skills是从财新网的《美国资产管理行业的历史性变革（上/下）》两篇文章中提炼出来的方法论。原文深入分析美国资管行业从1940年代至今的演变历程，并总结了可复用的方法论框架。
