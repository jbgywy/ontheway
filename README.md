# Skills Repository / 方法论Skills仓库

本仓库包含从《美国资产管理行业的历史性变革》研究中提炼的7个方法论Skills。这些Skills可以帮助企业进行战略规划、产品管理、组织发展等。

This repository contains 7 methodology skills extracted from the research on "Historical Changes in the US Asset Management Industry". These skills can help enterprises with strategic planning, product management, organizational development, etc.

## 📚 Skills列表 (Skills List)

### 战略规划类 (Strategic Planning)

1. **行业变革分析框架** (Industry Change Analysis Framework)
   - 分析任何行业历史性变革的系统化方法论
   - 适用于金融、科技、制造业等所有面临转型的行业
   - Systematic methodology for analyzing historical changes in any industry

2. **战略定位三维度** (Strategic Positioning Three Dimensions)
   - 帮助企业在变革环境中找到差异化定位
   - 避免同质化竞争
   - Help enterprises find differentiated positioning in changing environments

3. **竞争模式四分法** (Competition Mode Quadrant Method)
   - 帮助企业识别所在行业的竞争模式
   - 选择适合自己的竞争策略
   - Help enterprises identify industry competition modes and choose appropriate strategies

### 产品管理类 (Product Management)

4. **需求转化框架** (Demand Transformation Framework)
   - 将终端用户的隐性需求转化为可操作的产品功能
   - 适用于新产品开发、现有产品迭代优化
   - Transform implicit user needs into actionable product features

5. **可行性三时间轴** (Feasibility Three Time Horizons)
   - 评估战略举措的可行性
   - 按照短中长期时间轴分解实施路径
   - Evaluate feasibility of strategic initiatives and decompose implementation across short/medium/long-term

### 组织管理类 (Organizational Management)

6. **组织能力重组五维度** (Organizational Capability Reorganization Five Dimensions)
   - 帮助企业系统性评估并重构组织能力
   - 适应行业变革和战略转型
   - Systematically evaluate and reorganize organizational capabilities for industry changes

7. **科技投入优先级** (Technology Investment Priority)
   - 帮助企业在资源有限的情况下
   - 科学决定科技投入的优先顺序，最大化ROI
   - Scientifically determine technology investment priorities to maximize ROI

Each skill is an independent folder containing a `SKILL.md` file. You can:
1. Read directly to learn the methodology
2. Use the templates included in each skill
3. Apply the tools and frameworks recommended
4. Import into supporting systems (e.g., WorkBuddy)

## 📖 背景 (Background)

这些Skills是从财新网的《美国资产管理行业的历史性变革（上/下）》两篇文章中提炼出来的方法论。

原文深入分析美国资管行业从1940年代至今的演变历程，包括：
- 被动投资革命（指数基金、ETF的崛起）
- 费用压缩趋势（从1.5%降至0.03%）
- 投资者行为转型（从主动选股到资产配置）
- 资管公司应对策略（Vanguard、Fidelity、BlackRock案例）
- 四大竞争定位（被动巨头、主动精品、智能投顾、综合平台）

This methodology was extracted from two articles published by Caixin ("美国资产管理行业的历史性变革（上/下）"), which deeply analyze the evolution of the US asset management industry from the 1940s to present.

### Skills格式要求 (Skills Format Requirements)

每个Skill应包含以下部分 (Each skill should contain):
- **用途** (Purpose) - 这个Skill解决什么问题？
- **适用场景** (Applicable Scenarios) - 什么时候使用这个Skill？
- **方法论核心** (Core Methodology) - 具体的分析框架、工具、模板
- **输出模板** (Output Templates) - 标准化的交付物格式
- **应用案例** (Application Cases) - 真实或假设的案例
- **注意事项** (Precautions) - 使用时的常见陷阱
- **工具推荐** (Tool Recommendations) - 配套工具、软件
