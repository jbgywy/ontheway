# 战略定位三维度

## 用途
帮助企业或产品在变革环境中找到差异化定位，避免同质化竞争。

## 适用场景
- 行业变革期的企业战略重构
- 新产品/新业务线定位
- 面对颠覆性技术的应对策略
- 竞争格局重塑时的差异化选择

## 三维度模型

### 维度1：客户价值定位（Value Proposition）
**核心问题：我们为谁创造什么独特价值？**

| 定位选择 | 说明 | 案例 |
|---------|------|------|
| 成本领先 | 提供最低成本解决方案 | Vanguard（指数基金） |
| 差异化 | 提供独特功能/体验 | Apple（智能手机） |
| 聚焦细分 | 服务特定客群/场景 | 拼多多（下沉市场） |

**分析工具：价值曲线（Value Curve）**
```
绘制本企业 vs. 竞争对手在以下维度的得分（1-10分）：
- 价格
- 质量
- 便捷性
- 定制化
- 品牌
- 服务
```
找出价值空白点，这就是差异化机会。

### 维度2：能力边界定位（Capability Boundary）
**核心问题：我们能做好什么？什么应该外包/合作？**

**分析框架：VRIO模型**
- **V**aluable（有价值）：能否提升竞争优势？
- **R**are（稀缺性）：竞争对手是否难以获得？
- **I**nimitable（难以模仿）：是否有专利/网络效应/品牌护城河？
- **O**rganized（组织支持）：组织架构是否支持该能力发挥？

**决策矩阵：**
| 能力 | VRIO评估 | 战略选择 |
|------|-----------|----------|
| 核心能力 | VRIO均为"是" | 内部强化，构建护城河 |
| 重要但非核心 | VRIO部分"是" | 战略合作/合资 |
| 通用能力 | VRIO均为"否" | 外包 |

### 维度3：生态位定位（Ecosystem Niche）
**核心问题：我们在产业链/生态系统中处于什么位置？**

**生态位选择：**
1. **平台型**：构建生态系统，制定规则（如iOS、微信）
2. **互补型**：为平台提供互补产品/服务（如App开发者）
3. **垂直整合型**：控制产业链多个环节（如特斯拉）
4. **专业型**：聚焦单一环节，做到极致（如台积电）

**分析工具：生态系统地图**
```
绘制所在行业的生态系统地图：
- 上游供应商
- 平台/渠道
- 互补产品/服务
- 下游客户
- 监管机构
```
识别：
- 哪些环节利润最高？
- 哪些环节竞争最激烈？
- 哪些环节有被替代风险？

## 应用案例
- ✅ Vanguard：成本领先 + 被动投资能力 + 平台型生态位
- ✅ 苹果：差异化 + 软硬件一体化能力 + 平台型生态位
- ✅ 国内寿险公司：聚焦细分 + 资产负债管理能力 + 专业型生态位

## 注意事项
1. **定位不是一成不变的**：行业变革期，需要定期重新评估三维度
2. **避免"定位陷阱"**：
   - 定位过于宽泛（"为所有人提供所有价值"）
   - 定位与能力不匹配（"我们有这个能力吗？"）
   - 定位与生态系统冲突（"平台会允许我们这样做吗？"）
3. **量化验证**：所有定位选择必须有数据支撑，避免主观判断

## 工具推荐
- **战略地图**：平衡计分卡（BSC）工具绘制战略地图
- **价值曲线**：Excel/PowerPoint绘制价值曲线对比图
- **生态系统地图**：Miro/Lucidchart绘制生态系统可视化地图
